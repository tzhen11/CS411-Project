The primary goal of the website is to offer some opinions about travel destinations. Users will put in available time, and choose their preference about the destination. Destinations and their information, including hotel, flight, visa, historical weather, and featuring spots, will be presented on the website. If the user did not input any preference, the website would return some random places for the user.
